# CustomActionFlutterFlow

Este repositório contém ações personalizadas desenvolvidas para uso em projetos FlutterFlow, integradas com o backend **Xano**.  
O código foi originalmente criado para fins acadêmicos, como parte de atividades da faculdade.

## 🚀 Objetivo
Demonstrar como criar e integrar **Custom Actions** no FlutterFlow com suporte a banco de dados via Xano, permitindo maior flexibilidade e personalização além das funcionalidades nativas da plataforma.

## ⚠️ Aviso Importante
- Este código foi feito para **uso acadêmico** e pode não estar otimizado para produção.  
- **Todos os links externos devem ser removidos ou substituídos** conforme necessário antes de qualquer publicação ou uso fora do contexto da faculdade.  
- Caso utilize este repositório em projetos pessoais ou profissionais, revise e ajuste as dependências, credenciais e referências.

## 🛠️ Tecnologias Utilizadas
- [FlutterFlow](https://flutterflow.io/)  
- [Xano](https://xano.com/)
